# Agents

## Frontend Migration Agent
Purpose: Convert ASP.NET pages to Angular components. Inputs: .aspx, .ascx, markup, styles.

## Backend API Agent
Purpose: Extract business logic and create .NET Core services/controllers.

## Database Migration Agent
Purpose: Map DB schema and create EF Core migrations.

## Testing Agent
Purpose: Generate unit and integration tests for migrated modules.

## Review Agent
Purpose: Validate code quality, security, and performance.

## Agent Protocols
- Hand-offs, file formats, and validation checkpoints
